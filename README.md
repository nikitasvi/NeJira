## Run Locally  

Clone the project  

~~~bash  
  git clone https://github.com/nikitasvi/NeJira
~~~

Go to the project directory  

~~~bash  
  cd NeJira
~~~

Install dependencies  

~~~bash  
npm install
~~~

Start client  

~~~bash  
ng serve
~~~

Go to backend path 

~~~bash  
cd src/express
~~~

Start server  

~~~bash  
node server.js
~~~
